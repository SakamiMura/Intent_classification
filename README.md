# Intent Classification Project

Projekt klasyfikacji intencji użytkownika na zbiorze CLINC150.

## Technologie

- Python 3.10+
- scikit-learn
- pandas
- numpy
- nltk
- spacy

## Struktura projektu

- data/ — dane wejściowe
- models/ — zapisane modele
- src/ — kod projektu
- notebooks/ — notatniki eksperymentalne
